1 # Dockerfile for PetIBM-0.4
2 
3 The present directory contains the `Dockerfile` used to create the Docker image `petibm:0.4-GPU-OpenMPI-ubuntu`.
4 The Docker image is available on DockerHub; to pull the image:
5 
6 ```shell
7 docker pull barbagroup/petibm:0.4-GPU-OpenMPI-ubuntu
8 ```
9 
10 In this image, PetIBM (0.4) was installed along with its dependencies:
11 
12 * CUDA Toolkit 8.0
13 * OpenMPI 1.10.2
14 * PETSc 3.10.2
15 * AmgX 2.0 (commit [732338c](https://github.com/NVIDIA/AMGX/tree/732338c32e30ad87f9b71244346346f66fc3f735))
16 * AmgXWrapper 1.4
17 
18 Docker containers created out of this image can run PetIBM applications on either CPU or GPU devices.
19 Note that if you want to run a PetIBM application using GPUs, you have to create the container using the utility `nvidia-docker` (instead of `docker`):
20 
21 ```shell
22 nvidia-docker run -it barbagroup/petibm:0.4-GPU-OpenMPI-ubuntu /bin/bash
23 ```
24 
25 If you wish to re-build the image locally:
26 
27 ```shell
28 docker build --tag=mypetibm:latest --file=Dockerfile-0.4-GPU-OpenMPI-ubuntu .
29 ```
